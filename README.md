# ReadFlow Web 🌐📚

ReadFlow Web es la aplicación web del ecosistema ReadFlow, diseñada para la gestión y lectura de libros digitales desde el navegador. Esta versión complementa a la aplicación móvil y comparte la misma lógica de consumo de datos mediante Azure API Management, ofreciendo una experiencia organizada, visual y accesible para explorar la biblioteca, administrar libros y continuar lecturas.

## 🔗 Enlaces del proyecto

- **Aplicación web desplegada:** https://brave-sea-03b672010.2.azurestaticapps.net
- **Repositorio de la aplicación móvil:** https://github.com/Viktor1712/DesignReadAppProject

## 🚀 Características Principales

- **Biblioteca Digital:** Visualiza libros disponibles con su portada, autor, categoría y metadatos principales.
- **Exploración y búsqueda:** Permite buscar libros, autores y categorías desde la interfaz principal.
- **Continuar lectura:** Muestra libros en progreso para retomar la lectura desde el entorno web.
- **Administración de libros:** Incluye una vista para gestionar libros mediante operaciones CRUD simuladas con servicios mock.
- **Visualización de PDF:** Integración para lectura de documentos PDF dentro de la aplicación.
- **Consumo de API en Azure:** La aplicación consume servicios expuestos en Azure API Management.
- **Interfaz moderna:** Diseñada con una estructura clara, navegación lateral persistente y enfoque visual orientado a la usabilidad.

## 🛠️ Arquitectura y Tecnologías

El proyecto fue construido con tecnologías modernas de frontend web:

- **Lenguaje:** JavaScript
- **Framework UI:** React
- **Build Tool:** Vite
- **Routing:** React Router DOM
- **Visualización de PDF:** React PDF + pdfjs-dist
- **Backend/API:** Azure API Management
- **Hosting:** Azure Static Web Apps
- **CI/CD:** GitHub Actions
